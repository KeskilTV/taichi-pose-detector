# 🥋 TaiChi Pose Detector

Проект для анализа движений Тайцзи с использованием компьютерного зрения (MediaPipe) и OpenCV.  
Приложение обнаруживает позу человека на видео, отрисовывает скелет и позволяет оценивать технику выполнения форм.



## 🚀 Возможности

- **Обнаружение позы:** Автоматическое распознавание 33 ключевых точек тела.
- **Визуализация:** Отрисовка скелета поверх видео в реальном времени.
- **Гибкая настройка:** Цвета, толщина линий и параметры детекции через `config.py`.
- **Удобный интерфейс:** Выбор видеофайла через стандартное окно диалога.
- **Модульная архитектура:** Легкое расширение функционала (сравнение поз, AR, экспорт данных).

## 📋 Требования

- Python 3.8 или выше
- Операционная система: Windows / macOS / Linux
- Камера или видеофайл для обработки

## 🛠 Установка

1. **Клонируйте репозиторий:**
   Откройте терминал и выполните:
   git clone <URL-репозитория>
   cd taichi

2. **Создайте виртуальное окружение:**
   python -m venv venv

3. **Активируйте окружение:**
   Windows: venv\Scripts\activate
   macOS/Linux: source venv/bin/activate

4. **Установите зависимости:**
   pip install -r requirements.txt

## ▶️ Запуск

1. Поместите видеофайл в папку `input_videos/` (или выберите через диалог).
2. Запустите приложение:
   python main.py
3. Выберите видеофайл в появившемся окне.
4. Результат обработки сохранится в папке `output_videos/`.

## ⚙️ Конфигурация

Все основные настройки находятся в файле `config.py`:
- `POSE_SETTINGS`: Цвета и толщина отрисовки скелета.
- `POSE_DETECTION_SETTINGS`: Чувствительность и сложность модели MediaPipe.
- `INPUT_DIR` / `OUTPUT_DIR`: Папки для входных и выходных файлов.
